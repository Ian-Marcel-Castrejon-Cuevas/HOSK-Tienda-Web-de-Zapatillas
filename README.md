# HOSK – Tienda Web de Zapatillas

Sitio web desarrollado con **HTML, CSS, Bootstrap y JavaScript** que presenta una plataforma para la venta de zapatillas de diferentes marcas de manera fácil y segura.

## Descripción

El proyecto consiste en una página web para la tienda **HOSK**, donde los usuarios pueden conocer la empresa, explorar contenido visual mediante un slider interactivo y acceder al sistema para iniciar sesión y realizar compras.

## Funcionalidades

* Página principal con presentación de la tienda.
* Slider interactivo con imágenes promocionales.
* Sección **Nosotros** con información del equipo de trabajo.
* Acceso a un **sistema de login** para usuarios.
* Formulario de **contacto y soporte**.
* Integración con redes sociales.
* Diseño web responsivo.

## Tecnologías utilizadas

* HTML5
* CSS3
* Bootstrap
* JavaScript
* jQuery
